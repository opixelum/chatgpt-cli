# ChatGPT CLI

ChatGPT but in your terminal.

## Getting Started

### Prerequisites

1. Install `dotenv` & `openai` package:

   *Example using `pip`:*

   ```bash
   pip install dotenv
   ```

2. Copy `.env.example` into `.env`:

    ```bash
    cp .env.example .env
    ```
   
3. Fill `OPENAI_API_KEY` in `.env` with your OpenAI API key.

   ```text
    OPENAI_API_KEY=sk-...
    ```
   
### Usage

Run it!

 ```bash
 python main.py
 ```
